# Kiểm tra tuân thủ đội mũ bảo vệ với YOLOv10

Chương trình phát hiện các nhân viên có đeo mũ bảo vệ trong công trường hay không?

• Input: Một tấm ảnh.

• Output: Tọa độ (bounding box) của các nhân viên và phần mũ bảo hiểm.
